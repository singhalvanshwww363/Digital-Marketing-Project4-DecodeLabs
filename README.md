## 🧠 Key Concepts Covered

### Analytics Funnel
```
AWARENESS → ENGAGEMENT → CONSIDERATION → CONVERSION
73,630 reach  4.2% ER    2,581 visits    73 enquiries
```

### Engagement Rate Formula
```
ER = [(Likes + Comments + Shares + Saves) / Reach] × 100
2026 Benchmark: ER > 5.2% = Above Average
```

### Weighted Algorithmic Currency
| Tier | Action | Weight | Why |
|------|--------|--------|-----|
| Tier 3 | Shares | 5.0 | Viral potential |
| Tier 2 | Saves | 3.0 | Evergreen intent |
| Tier 1 | Likes | 1.0 | Quick approval |

## 💡 Top Data-Driven Insights

1. **Reels outperform static posts** by 3x — 6.2% vs 2.1% engagement
2. **LinkedIn carousels Tue/Wed 8-9am** get highest CTR at 4.8%
3. **Question CTAs** generate 4x more comments than posts without CTA
4. **Data-backed posts** get 2.5x more shares than opinion posts
5. **Save behaviour:** Career tips get 3x more saves — high bookmark intent
